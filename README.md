## 🎯 CORE FEATURES

<table>
<tr>
<td width="50%">

### 📸 **AGRI-VISION DIAGNOSTICS**
- 🔍 **Real-time Camera Capture:** Snap crop pictures directly with webcams or mobile cameras.
- 🌱 **Foliage Disease Detection:** Identify plant species and diagnose specific leaf/stem issues.
- 📋 **Diagnostics Report:** Instant formatted botanical diagnostic breakdown.
- ⚡ **Local Processing:** Instant base64 encoding and direct API calls.

</td>
<td width="50%">

### 🤖 **AGRONOMIST AI CHAT**
- 💬 **Context-Aware Memory:** Conversational chatbot that remembers preceding thread messages.
- 🎙️ **Voice Typing:** Speak to the assistant to dictate messages (STT).
- 🔊 **Read-Aloud Answers:** Play back bot diagnostic responses using TTS.
- 🌿 **Expert Agricultural Insights:** Tailored crop fertilizing, soil pH, and watering care tips.

</td>

## ⚡ LIGHTNING FAST SETUP

```bash
# 1️⃣ Clone the repository
git clone https://github.com/deepakrajjs-29/VI-Assistant.git

# 2️⃣ Navigate to directory
cd VI-Assistant

# 3️⃣ Install dependencies
npm install

# 4️⃣ Run development server
npm run dev
```

## 🔒 SECURITY & CONFIGURATION

This project runs **entirely client-side**. 
- Your Gemini API Key is stored securely on your browser’s `localStorage` and never uploaded to any remote server.
- Obtain a free API Key from [Google AI Studio](https://aistudio.google.com/).
